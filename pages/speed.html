三步快速开始

按顺序完成即可使用
1
安装脚本管理器
浏览器安装 Tampermonkey（Chrome/Edge）或国内用户可用 ScriptCat 脚本猫，用于运行 Userscript。
2
安装 LinkSwift 脚本
安装 LinkSwift 正式版（国内源）。打开对应网盘网页，勾选文件后点击「下载助手」。
3
配置下载器
安装 Motrix 或 Aria2，在脚本中选择 RPC / API 推送，即可多线程下载。
4
分享资源给用户
需要分发链接时，用本站 链接生成器 制作二维码分享页，降低打开门槛。
