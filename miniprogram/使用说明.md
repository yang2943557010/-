# 网盘资源分享小程序使用说明

## 快速开始

### 1. 准备工作

1. 注册微信小程序账号：https://mp.weixin.qq.com/
2. 下载微信开发者工具：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 导入项目

1. 打开微信开发者工具
2. 点击"导入项目"
3. 选择 `miniprogram` 文件夹
4. 填入你的小程序AppID
5. 点击"导入"

### 3. 配置项目

#### 修改AppID

打开 `project.config.json` 文件，将 `appid` 修改为你的小程序AppID：

```json
{
  "appid": "your-appid-here",
  "projectname": "网盘资源分享小程序"
}
```

#### 修改域名（可选）

如果你有自己的服务器，可以修改 `utils/url-handler.js` 中的域名：

```javascript
const baseUrl = 'https://your-domain.com/index.html';
```

### 4. 运行项目

1. 点击"编译"按钮
2. 在模拟器中查看效果
3. 使用"预览"功能在真机上测试

## 功能说明

### 主页面功能

1. **显示资源信息**
   - 资源名称
   - 提取码
   - 网盘类型识别

2. **二维码显示**
   - 自动生成二维码
   - 支持扫码访问

3. **广告弹窗**
   - 可设置广告文字
   - 可设置跳转时长
   - 支持跳过功能

4. **页面风格**
   - 支持多种主题样式
   - 可自定义背景

### 生成功能

1. **生成分享链接**
   - 输入网盘链接
   - 设置资源名称
   - 设置提取码
   - 选择页面风格

2. **生成二维码**
   - 自动生成二维码图片
   - 支持保存到相册

## 注意事项

1. **链接格式**
   - 必须是完整的网盘分享链接
   - 支持带提取码的链接

2. **二维码生成**
   - 小程序中生成的二维码是简化版
   - 如需完整二维码，需要集成第三方库

3. **跳转功能**
   - 小程序无法直接跳转外部链接
   - 会复制链接到剪贴板，提示用户在浏览器中打开

4. **广告功能**
   - 广告文字支持换行
   - 广告时长限制1-5秒

## 常见问题

### Q: 为什么无法生成二维码？
A: 小程序中生成二维码需要使用专门的库，当前版本使用占位符。如需完整功能，可以集成 `weapp-qrcode` 等第三方库。

### Q: 如何添加新的网盘支持？
A: 在 `utils/disk-config.js` 中添加新的网盘配置即可。

### Q: 如何修改页面样式？
A: 修改对应的 `.wxss` 文件，支持自定义主题颜色。

### Q: 如何部署到线上？
A: 1. 在微信开发者工具中点击"上传"
   2. 在微信公众平台提交审核
   3. 审核通过后发布
