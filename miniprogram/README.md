# 网盘资源分享微信小程序

## 项目简介

这是一个完全还原原网页样式的微信小程序，用于网盘资源分享。支持多种网盘类型，包含二维码显示、广告弹窗、页面风格切换等功能。

## 功能特性

- ✅ 支持20+种网盘类型自动识别
- ✅ 二维码生成和显示
- ✅ 广告弹窗功能（可配置文字和时长）
- ✅ 10种页面风格切换
- ✅ 提取码显示和复制
- ✅ 完全还原原网页样式
- ✅ 响应式设计，适配移动端和桌面端

## 快速开始

### 1. 配置AppID

打开 `project.config.json` 文件，将 `appid` 修改为你的小程序AppID：

```json
{
  "appid": "your-appid-here"
}
```

### 2. 导入项目

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `miniprogram` 文件夹
4. 点击"导入"

### 3. 运行项目

1. 点击"编译"按钮
2. 在模拟器中查看效果
3. 使用"预览"功能在真机上测试

## 注意事项

1. **域名配置**：如需线上运行，需要在微信公众平台配置服务器域名
2. **二维码生成**：当前使用占位符实现，如需完整功能可集成第三方库
3. **跳转功能**：小程序无法直接跳转外部链接，会复制链接到剪贴板
4. **资源文件**：需要将原网页的assets文件复制到小程序目录

## 开发说明

### 修改配置

编辑 `config.js` 文件可以修改：
- 服务器域名
- 默认广告时长
- 默认页面风格
- 其他配置项

### 添加网盘支持

在 `utils/disk-config.js` 中添加新的网盘配置：

```javascript
newDisk: {
  name: '新网盘',
  keywords: ['newdisk.com'],
  logo: '/assets/logos/newdisk.png',
  guide: '/assets/images/guides/newdisk-guide.png',
  appName: '新网盘APP',
  color: '#ff0000'
}
```

### 自定义样式

修改对应的 `.wxss` 文件可以自定义页面样式。

## 常见问题

### Q: 如何获取AppID？
A: 登录微信公众平台，进入"开发" -> "开发设置"，找到"AppID(小程序ID)"。

### Q: 如何配置服务器域名？
A: 登录微信公众平台，进入"开发" -> "开发设置"，在"服务器域名"中添加你的域名。

### Q: 如何添加新的网盘支持？
A: 在 `utils/disk-config.js` 中添加新的网盘配置即可。

### Q: 如何修改页面风格？
A: 在生成链接时选择不同的页面风格，或修改 `config.js` 中的默认风格。
