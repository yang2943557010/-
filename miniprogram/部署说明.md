# 网盘资源分享小程序部署说明

## 部署前准备

### 1. 注册小程序账号
1. 访问微信公众平台：https://mp.weixin.qq.com/
2. 点击"立即注册"
3. 选择"小程序"类型
4. 填写邮箱和密码
5. 完成邮箱验证
6. 选择主体类型（个人/企业）
7. 填写主体信息
8. 完成注册

### 2. 获取AppID
1. 登录微信公众平台
2. 进入"开发" -> "开发设置"
3. 找到"AppID(小程序ID)"
4. 复制AppID

### 3. 下载开发者工具
1. 访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 下载对应系统的版本
3. 安装并登录

## 部署步骤

### 步骤1：配置项目

#### 修改AppID
打开 `project.config.json` 文件：
```json
{
  "appid": "你的AppID",
  "projectname": "网盘资源分享小程序"
}
```

#### 修改域名（可选）
如果你有自己的服务器，修改 `config.js`：
```javascript
const config = {
  baseUrl: 'https://你的域名.com',
  sharePagePath: '/index.html',
  // ... 其他配置
};
```

### 步骤2：导入项目

1. 打开微信开发者工具
2. 点击"导入项目"
3. 选择 `miniprogram` 文件夹
4. 填入AppID
5. 点击"导入"

### 步骤3：配置服务器域名

#### 登录微信公众平台
1. 进入"开发" -> "开发设置"
2. 找到"服务器域名"
3. 点击"修改"

#### 添加域名
在"request合法域名"中添加：
```
https://your-domain.com
https://wx.251800.xyz
```

在"uploadFile合法域名"中添加（如果需要）：
```
https://your-domain.com
```

在"downloadFile合法域名"中添加（如果需要）：
```
https://your-domain.com
```

### 步骤4：上传代码

1. 在微信开发者工具中点击"上传"
2. 填写版本号（如：1.0.0）
3. 填写项目备注
4. 点击"上传"

### 步骤5：提交审核

1. 登录微信公众平台
2. 进入"版本管理"
3. 找到刚上传的版本
4. 点击"提交审核"
5. 填写审核信息
6. 等待审核（通常1-7天）

### 步骤6：发布上线

1. 审核通过后
2. 进入"版本管理"
3. 点击"发布"
4. 选择发布范围（全部用户/部分用户）
5. 确认发布

## 配置说明

### 1. 服务器配置

#### 域名要求
- 必须是已备案的域名
- 必须支持HTTPS
- 不能使用IP地址
- 不能使用端口号

#### SSL证书
- 必须有效
- 不能是自签名证书
- 建议使用Let's Encrypt免费证书

### 2. 小程序配置

#### app.json配置
```json
{
  "pages": [
    "pages/index/index",
    "pages/generator/generator"
  ],
  "window": {
    "backgroundTextStyle": "light",
    "navigationBarBackgroundColor": "#f3f6fb",
    "navigationBarTitleText": "资源分享",
    "navigationBarTextStyle": "black"
  }
}
```

#### 权限配置
在 `app.json` 中添加需要的权限：
```json
{
  "permission": {
    "scope.userLocation": {
      "desc": "你的位置信息将用于小程序位置接口的效果展示"
    }
  }
}
```

### 3. 功能配置

#### 广告功能配置
在 `config.js` 中配置：
```javascript
const config = {
  enableAd: true,           // 是否启用广告
  defaultAdDuration: 3,     // 默认广告时长
  // ... 其他配置
};
```

#### 页面风格配置
在 `config.js` 中配置：
```javascript
const config = {
  defaultTemplate: 'default',  // 默认风格
  // 可选: default, gradient, sunset, ocean, forest, cherry, midnight, aurora, candy, card
};
```

## 常见问题

### Q1: 域名配置错误
**问题**：提示"域名不在以下 request 合法域名列表中"

**解决**：
1. 检查域名是否已备案
2. 检查域名是否支持HTTPS
3. 检查域名是否已添加到微信公众平台
4. 等待域名配置生效（通常几分钟）

### Q2: 上传失败
**问题**：上传代码时失败

**解决**：
1. 检查AppID是否正确
2. 检查网络连接
3. 检查代码是否有语法错误
4. 重启开发者工具

### Q3: 审核被拒
**问题**：审核不通过

**常见原因**：
1. 功能不完整
2. 内容违规
3. 用户体验差
4. 存在安全风险

**解决**：
1. 仔细阅读审核反馈
2. 修改问题代码
3. 重新提交审核

### Q4: 无法生成二维码
**问题**：二维码生成失败

**解决**：
1. 检查canvas组件是否正确
2. 检查二维码库是否集成
3. 检查权限配置
