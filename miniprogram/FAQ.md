# 网盘资源分享小程序常见问题解答

## 基础问题

### Q1: 什么是网盘资源分享小程序？
**A**: 这是一个将网盘资源分享页面转换为微信小程序的项目，支持多种网盘类型，可以生成二维码和分享链接。

### Q2: 支持哪些网盘？
**A**: 支持20+种网盘，包括：
- 国内：百度网盘、夸克网盘、阿里云盘、迅雷云盘、115网盘、蓝奏云、天翼云盘、腾讯微云、坚果云等
- 国际：OneDrive、Google Drive、Dropbox、MEGA等

### Q3: 需要服务器吗？
**A**: 可选。如果只是本地使用，不需要服务器。如果需要线上部署，需要一个支持HTTPS的服务器。

### Q4: 需要付费吗？
**A**: 小程序本身免费，但需要：
- 微信小程序账号（免费）
- 服务器（可选，可能需要付费）
- 域名（可选，可能需要付费）

## 安装问题

### Q5: 如何安装小程序？
**A**: 
1. 下载微信开发者工具
2. 导入 `miniprogram` 文件夹
3. 填入AppID
4. 点击编译运行

### Q6: 提示"AppID错误"怎么办？
**A**: 
1. 检查AppID是否正确
2. 确认AppID是否已激活
3. 检查是否填写了正确的AppID

### Q7: 提示"域名不在合法列表中"怎么办？
**A**: 
1. 登录微信公众平台
2. 进入"开发" -> "开发设置"
3. 添加服务器域名
4. 等待配置生效

### Q8: 如何获取AppID？
**A**: 
1. 登录微信公众平台：https://mp.weixin.qq.com/
2. 进入"开发" -> "开发设置"
3. 找到"AppID(小程序ID)"
4. 复制AppID

## 功能问题

### Q9: 为什么无法生成二维码？
**A**: 当前版本使用占位符实现，如需完整功能：
1. 集成 `weapp-qrcode` 库
2. 或使用其他二维码生成库
3. 参考第三方库文档进行集成

### Q10: 如何修改服务器域名？
**A**: 
1. 打开 `config.js` 文件
2. 修改 `baseUrl` 配置
3. 重新编译小程序

### Q11: 如何添加新的网盘支持？
**A**: 
1. 打开 `utils/disk-config.js`
2. 添加新的网盘配置
3. 包括：name、keywords、logo、guide、appName、color

### Q12: 如何修改页面样式？
**A**: 
1. 找到对应的 `.wxss` 文件
2. 修改样式代码
3. 重新编译查看效果

### Q13: 广告功能如何使用？
**A**: 
1. 生成链接时填写广告文字
2. 设置广告时长（1-5秒）
3. 用户打开链接时会先看到广告
4. 广告结束后自动跳转

### Q14: 如何切换页面风格？
**A**: 
1. 在生成器页面选择"页面风格"
2. 选择喜欢的风格
3. 生成链接后会应用该风格

## 部署问题

### Q15: 如何部署到线上？
**A**: 
1. 在微信开发者工具中点击"上传"
2. 登录微信公众平台
3. 提交审核
4. 审核通过后发布

### Q16: 审核需要多长时间？
**A**: 通常1-7个工作日，具体时间取决于：
- 提交材料的完整性
- 小程序功能的复杂度
- 审核队列的长度

### Q17: 审核被拒怎么办？
**A**: 
1. 仔细阅读审核反馈
2. 根据反馈修改问题
3. 重新提交审核
4. 如有疑问可申诉

### Q18: 如何更新已发布的小程序？
**A**: 
1. 修改代码
2. 在开发者工具中上传
3. 提交审核
4. 审核通过后发布更新

## 技术问题

### Q19: 如何调试小程序？
**A**: 
1. 使用微信开发者工具的调试功能
2. 查看Console面板的日志
3. 使用Network面板查看网络请求
4. 使用AppData面板查看数据

### Q20: 如何查看错误日志？
**A**: 
1. 在开发者工具中打开Console面板
2. 查看红色错误信息
3. 根据错误信息定位问题

### Q21: 如何优化性能？
**A**: 
1. 减少setData调用次数
2. 使用分包加载
3. 优化图片资源
4. 使用缓存机制

### Q22: 如何处理网络请求失败？
**A**: 
1. 检查网络连接
2. 检查域名配置
3. 检查服务器状态
4. 添加错误处理代码

## 故障排除

### Q27: 小程序无法启动
**检查项**：
1. AppID是否正确
2. 代码是否有语法错误
3. 开发者工具是否最新版本
4. 网络连接是否正常

### Q28: 页面显示空白
**检查项**：
1. WXML文件是否正确
2. 数据是否正确绑定
3. 样式是否正确
4. 控制台是否有错误

### Q29: 功能不工作
**检查项**：
1. JS代码是否有错误
2. 数据是否正确传递
3. 权限是否正确配置
4. 网络请求是否成功

### Q30: 样式显示异常
**检查项**：
1. WXSS文件是否正确
2. 选择器是否正确
3. 单位是否正确
4. 是否有样式冲突
